## おかしな文 5つ（Night3入口パズル）

Night2後の追加ページに散りばめられた5つのおかしな文。各文の頭文字が太字カタカナで強調されており、繋げると「ロゴゴカイ」→「ロゴ五回」→ 公式サイトトップのタイトルロゴを5回クリック → Night3入口出現。

### 配置ルール
- 各おかしな文は600〜1000字の記事本文の中盤に配置
- **改行で独立させず、前後の文と同じ段落内に溶け込ませる**
- 文法は正常だが、前後の文脈から浮いている（内容の温度差がある）
- 頭文字はカタカナ＋ひらがな（「ゴご」「ゴまかし」「カならず」「イつも」）のパターン。太字のカタカナがさりげなく視覚的に引っかかる程度

### 5つのおかしな文と配置先

**① 「ロ」 → A-2（収録レポート #047）**
配置段落（企画監修の方の描写の中に埋め込む）：
> この日は珍しく、企画監修の方がスタジオにいらしていました。収録の様子を後ろから静かにご覧になっていましたが、スタッフの間でも普段お見かけしない方なので少し緊張しました。特に何か指示があったわけではなく、**ロ**うかの奥から誰かの声が聞こえた気がして振り返りましたが、誰もいませんでした。収録が終わるとその方もそのまま帰られました。

**② 「ゴ」 → A-6（収録レポート #189）**
配置段落（子ども参加収録の描写の中に埋め込む）：
> うっきちが子どもたちの手紙をこっそり覗き見して「えー、そんなのほしいの？」とツッコミを入れるシーンでは、翔太くんが本気で手紙を隠そうとしてくれて、とても微笑ましい光景でした。**ゴ**ご4時を過ぎると、スタジオの空気が変わる気がします。収録は夕方前に無事終了し、帰り際に子どもたちからキャストとスタッフ全員にクリスマスカードのプレゼントがありました。

**③ 「ゴ」 → B-1（山本Pブログ）**
配置段落（制作裏話の中に埋め込む）：
> 特にキツネ団長とかあ博士の掛け合いはテンポが命なので、鈴木さんと高橋さんは本番前に必ず二人で合わせをやっています。**ゴ**まかしきれないことがある、とふと思うことがあります。それでも、毎週土曜の朝に子どもたちが「今日はどんなお話かな」と楽しみにしてくれている。

**④ 「カ」 → B-2（田中脚本ブログ）**
配置段落（キャラの口調の話の後に埋め込む）：
> キツネ団長は「〜だぞ！」、かあ博士は「〜なのだ」、ぽたまるは「〜でち」、うっきちは「〜だもん」。この語尾だけで誰が話しているかわかるようにしています。**カ**ならず記録は残しておくべきだと、あの方がよく言っていました。最近、4年分の脚本を読み返す機会がありました。

**⑤ 「イ」 → C-1（撮影現場レポート1：スタジオセット）**
配置段落（ダンスフロアの解説の中に埋め込む）：
> 床面はピンクと白の市松模様で、照明が当たるとキラキラ光る加工がされています。**イ**つもとは違う手順で収録が行われた日がありました。何のためだったのかは聞けませんでした。そして一番奥が「うっきちの小道具倉庫」。

### 太字の実装
HTMLでは以下のように実装：
```html
<strong>ロ</strong>うかの奥から誰かの声が聞こえた気がして振り返りましたが、誰もいませんでした。
```
CSSで太字のスタイルが目立ちすぎないよう調整。通常の太字と同じ見た目にする。

### A-2：収録レポート #047「キツネ団長の宝さがし」
- パス：reports/recording-047.html
- 収録日：1992年2月17日
- 署名：（広報部 鈴木）
- 文字数：600〜1000字
- おかしな文：①「**ロ**うかの奥から〜」（上記参照）
- 仕込み：企画監修の方がスタジオに来ていた描写。スタッフも普段見かけない人物＝staff.htmlの塗りつぶし人物との接続
- テキスト方針：宝さがし回の和やかな収録風景。美術チームの仕込み、操演のアドリブなど楽しい雰囲気の中に、企画監修の訪問が自然に紛れ込む

**サンプルテキスト（これをベースに調整）：**

```
番組が始まって1年が経とうとしています。今回は「キツネ団長の宝さがし」の収録レポートです。

この回は、キツネ団長がスタジオ中に隠された宝箱を探し回るという冒険仕立てのお話です。セットのあちこちに木箱や小さなチェストが隠されていて、美術チームの吉田さんがいつもより早い朝6時からセットの仕込みに入っていました。「宝箱を開ける瞬間に子どもたちが本当にワクワクしてくれるように」と、箱ごとに中身のキラキラした演出を変えているこだわりようです。

本番では、かあ博士が宝の地図を読み上げるシーンで、操演の高橋さんが眼鏡を光らせるタイミングをぴったり合わせて大成功。キツネ団長が宝箱を見つけるたびに懐中時計を掲げて「大発見だ！」と決めポーズをとる演出は、鈴木さんのアドリブだったそうです。

この日は珍しく、企画監修の方がスタジオにいらしていました。収録の様子を後ろから静かにご覧になっていましたが、スタッフの間でも普段お見かけしない方なので少し緊張しました。特に何か指示があったわけではなく、<strong>ロ</strong>うかの奥から誰かの声が聞こえた気がして振り返りましたが、誰もいませんでした。収録が終わるとその方もそのまま帰られました。

収録後はスタッフ全員でロケハンの打ち合わせ。春の特番に向けて屋外ロケの候補地を話し合いました。2年目に向けて番組がますます盛り上がっていきそうで、楽しみです。

（広報部 鈴木）

これからもこの4人と一緒に、子どもたちが笑顔になれるお話を書いていきたいと思います。

### C-1：撮影現場レポート1「スタジオセット・遊園地各エリア解説」
- パス：reports/set-tour.html
- 署名：（広報部 鈴木）
- 文字数：600〜1000字
- おかしな文：⑤「**イ**つもとは違う手順で〜」（上記参照）
- 仕込み：なし（おかしな文のみ）

**サンプルテキスト（これをベースに調整）：**

```
今回は、木漏れ日遊園地のスタジオセットをご紹介します。

番組の収録は、きらら放送の第3スタジオで行われています。約200平米のスタジオ内に、遊園地をモチーフにした5つのエリアが設けられています。

## 技術仕様

### Night2クリアフラグ管理
- localStorage に `night2_clear: true` が設定されている場合のみ、Night2後ページが表示される
- 直接URLアクセスでもフラグチェックを行い、未クリアならリダイレクト

### デザイン
- 既存の公式アーカイブCSS（official.css）をベースに、Night2後ページ専用の追加CSS（official-restored.css）を作成
- ヘッダー・フッター・ナビゲーションは全ページ共通（既存と同じ）
- **Night2後ページは「復元されたアーカイブデータ」という設定のため、既存ページとは微妙にCSSが異なる**

### Night2後ページ専用CSS（official-restored.css）の差分

既存ページとの違いは以下の3点のみ。差は「言われてみれば違う」程度に抑える。

**1. 背景色がわずかに暖色寄り**
- 既存：空色ベース（例：#e8f4f8 系）
- Night2後：ほんの少しだけ暖色に振る（例：#f0ece4 系）
- 「古いデータが少し褪色した」印象

**2. フォントが少し古め**
- 既存：ゴシック系優先（例："Rounded Mplus 1c", sans-serif）
- Night2後：明朝系を先に指定（例："游明朝", "Yu Mincho", "HG明朝E", serif, sans-serif）
- 本文のみ。見出し・ナビゲーションはそのまま

**3. 全体にわずかにセピアフィルター**
- bodyまたはmainに `filter: sepia(0.05)` 程度を適用
- ヘッダー・ナビゲーションには適用しない（共通部分は通常のまま）
- 値が大きすぎると不自然なので 0.03〜0.08 の範囲で調整

**実装方法：**
```html
<!-- Night2後ページのhead内 -->
<link rel="stylesheet" href="css/official.css">
<link rel="stylesheet" href="css/official-restored.css">
```
official-restored.cssで上記3点のみオーバーライドする。

### 画像
- キャラクター画像は既存の /images/characters/*_normal.png を使用
- キャラ拡張ページではキャラ画像を大きく配置
- 画像のドラッグ・右クリック保存を無効化（既存の設定を踏襲）
