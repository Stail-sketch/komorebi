### CSS — official-restored.css リニューアル
⚠️ Night2後の既存ページにも適用される変更。official-restored.cssを上書き更新する。

コンセプト：「壊れたデータから復元された古いウェブページ」
CRTモニターで古いアーカイブを読み出しているような表示。

#### 基本スタイル
- 背景：Night1後ページ（official.css）の明るいポップな色調から一変。暗めのオフホワイト〜薄グレー（#f0ece4 程度）
- 本文フォント：monospace系を混ぜる。データ復元っぽさ
- 文字色：完全な黒ではなく、やや褪せたダークグレー（#2a2a2a）
- リンク色：彩度を落とした青（#4a6a8a 程度）

#### スキャンライン効果
ページ全体にCRTスキャンラインのオーバーレイを常時表示：

```css
/* スキャンラインオーバーレイ */
body::after {
  content: '';
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  pointer-events: none;
  z-index: 9998;
  background: repeating-linear-gradient(
    0deg,
    transparent,
    transparent 2px,
    rgba(0, 0, 0, 0.03) 2px,
    rgba(0, 0, 0, 0.03) 4px
  );
}
```

※ opacity 0.03 程度で「うっすら見える」レベル。読みにくくならないこと。
※ pointer-events: none で操作を妨げない。
※ z-indexは高いが、Night4入口演出のoverlayより低くすること。

#### 微弱なちらつきアニメーション（オプション）
ページ全体がごくわずかに明滅する。気づくか気づかないかのレベル：

```css
@keyframes flicker {
  0%, 95%, 100% { opacity: 1; }
  96% { opacity: 0.98; }
  97% { opacity: 1; }
  98% { opacity: 0.97; }
}

body {
  animation: flicker 8s infinite;
}
```

#### テキストのにじみ効果
本文テキストにごく微弱なテキストシャドウを付けて、CRT表示のにじみを再現：

```css
.restored-content p,
.restored-content li,
.restored-content td {
  text-shadow: 0.3px 0 0 rgba(0, 50, 100, 0.08),
               -0.3px 0 0 rgba(100, 0, 0, 0.08);
}
```

※ RGBの色ずれ風。ごくわずかに赤と青がズレている程度。

#### Night1後ページ（official.css）との差別化
- Night1後：元の公式サイトそのもの。ポップで明るい。変化なし
- Night2後以降：official-restored.css。「壊れたデータから復元した」見た目。暗め＋スキャンライン＋ちらつき

この差が「復元されたアーカイブデータ」という設定をプレイヤーに体感させる。

#### Night4入口の実装仕様

解答入力欄はHTMLの `<input>` ＋ ボタンで実装。

```html
<div class="quiz-answer">
  <label>こたえ →</label>
  <input type="text" id="quiz-answer-input" placeholder="" />
  <button id="quiz-submit-btn">おうぼする！</button>
</div>
```

JavaScript判定：
```javascript
document.getElementById('quiz-submit-btn').addEventListener('click', function() {
  const answer = document.getElementById('quiz-answer-input').value.trim();
  
  // Night4入口判定
  if (answer === 'みつけてね') {
    // Night3クリア済みであること確認
    if (localStorage.getItem('night3_cleared') === 'true') {
      // 演出開始
      triggerNight4Entrance();
    }
  }
  
  // ダミー問題の正解（何も起きない）
  // 08=く, 32=み, 25=の, 12=し, 45=ろ → 「くみのしろ」
  if (answer === 'くみのしろ') {
    // 何も起きない、または「せいかい！おうぼありがとう♪」的なメッセージ
  }
});

function triggerNight4Entrance() {
  // 1. 画面フリーズ演出
  document.body.style.pointerEvents = 'none';
  
  // 2. ノイズ化エフェクト（CSSアニメーション）
  document.body.classList.add('glitch-effect');
  
  // 3. 2秒後にメッセージ表示
  setTimeout(function() {
    const overlay = document.createElement('div');
    overlay.className = 'night-entrance-overlay';
    overlay.innerHTML = '<p>ー おめでとう。みつけてくれたね。ー</p>';
    document.body.appendChild(overlay);
    
    // 4. さらに2〜3秒後にNight4へ遷移
    setTimeout(function() {
      window.location.href = '（Night4のパス）';
    }, 2500);
  }, 2000);
}
```
