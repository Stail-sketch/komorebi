> FUNCTIONAL DESCRIPTION

UNIT-01 monitors temporal patterns through an integrated clock mechanism. The internal chronometer requires periodic calibration (winding) to maintain accurate data collection intervals.

When the clock mechanism winds down, UNIT-01's monitoring range expands as it seeks recalibration. Subjects within proximity will be targeted for direct interaction.

> BEHAVIORAL SPECIFICATIONS

- Default state: Stationary monitoring from primary position

- Clock degradation triggers movement toward control room

- Recalibration method: Manual clock winding resets monitoring range

- UNIT-01 response to winding: Immediate return to default position

> PATTERN OUTPUT

UNIT-01 embeds temporal markers in broadcast signal. These markers synchronize with subject circadian responses. Extended exposure results in DATA RESTRICTED.
