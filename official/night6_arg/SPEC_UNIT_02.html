> FUNCTIONAL DESCRIPTION

UNIT-02 captures high-resolution visual data of subjects during pattern exposure sessions. The integrated lens array records both visible-spectrum imagery and CLASSIFIED wavelength responses.

Data collected is cross-referenced with behavioral analysis module for real-time response evaluation.

- Phase 2: Room 6 (Corridor transit)

- Phase 3: Room 7 or Room 8 (Secondary observation)

- Phase 4: Door approach (Direct subject interaction)
