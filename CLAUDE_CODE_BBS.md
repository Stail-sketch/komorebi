# Claude Code 実装指示書：掲示板（5スレッド）

## 概要

ファンサイト掲示板の5スレッドを実装する。
2003年頃の日本のWeb掲示板（2ch風のシンプルな掲示板）を再現。

**重要な設計原則：**
- スレッド1〜3、5は普通の掲示板。懐かしく温かいトーン
- **スレッド4だけが異質**。ここがNight1へのメイン導線であり、時系列異常がある
- 掲示板は2003年で止まっている。新規書き込み不可
- スレッド4の>>1だけが2026年の日付（他は全て2003年以前）

- >>1と>>7だけが2026年。他は全部2003年。
- 2026年の書き込みに2003年の住人が「普通に」返信している
- **この異常について作中で一切説明しない**
- CSSで日付を特別に強調しない。気づくかどうかはプレイヤー次第
- ただし日付は全レスで同じフォントサイズ・色で表示する（隠さない）

#### 2. URLの実装

**URLの表示方式について：**
- URLは █（黒塗り文字）で表示する。実際のURLテキストは見せない
- ただしリンクとしてクリック可能
- ホバー時にブラウザのステータスバーにリンク先が表示されるのはOK（プレイヤーへのヒント）
- クリックするとNight1のゲームページに遷移する

#### 3. >>8以降の「返信なし」の表現

- 書き込みフォームは無効化状態で表示

#### 4. >>1（業くん）のレスの特徴
- 丁寧語を使う（「すみません」「見れたんですけど」）
- 深夜3時台に書き込んでいる（眠れない＝フラッシュバックの影響を示唆）
- 不安を感じつつも調べている
- 2回目の書き込み（>>7）で「自分だけ見れるっておかしいですよね」と言っている
- この後返信が途絶える（植物状態）

- 不気味な装飾は一切しない
